# Defunct: Delete data

This function is deprecated. Use
[`wb_clean_sheet()`](https://janmarvin.github.io/openxlsx2/reference/wb_clean_sheet.md).

## Usage

``` r
delete_data(wb, sheet, cols, rows)
```

## Arguments

- wb:

  workbook

- sheet:

  sheet to clean

- cols:

  numeric column vector

- rows:

  numeric row vector
